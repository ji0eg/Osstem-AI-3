## 2. 필요한 개념 키워드 (넓고 얕게)

### 🐍 Python 기초
- `pandas` — 엑셀처럼 표 데이터를 다루는 도구
- `openpyxl` — 엑셀 파일을 읽고 쓰는 도구
- `datetime` — 날짜 계산 (전월, 전분기 자동 계산)
- 조건문 / 반복문 — 이상 항목 자동 필터링의 기본

### 📊 데이터 처리
- 데이터 정합성 검증 — 숫자가 빠지거나 중복되지 않았는지 확인
- 피벗(Pivot) — 행·열을 바꿔 집계하는 방법 (엑셀 피벗테이블과 동일 개념)
- 전기 대비 비교 — 같은 기간 작년 수치와 나란히 놓는 것

### 📄 문서 자동화
- 템플릿(Template) — 미리 만들어둔 양식에 값만 채워 넣는 방식
- `python-docx` — 워드 파일 자동 생성 도구
- f-string — 문자열 안에 숫자를 끼워넣는 파이썬 문법

### 🏦 회계/공시 연계
- ERP 데이터 추출 — CSV, 엑셀 형태로 내보내기
- DSD (전자공시 작성 도구) — DART에 제출할 서류를 작성하는 시스템
- 계정과목 코드 — 자동화 시 계정을 구분하는 기준값
