# Osstem-AI-3 · 환율 데이터 수집 프로그램

한국은행에서 날짜별 환율(달러, 유로 등)을 자동으로 가져와서
엑셀 파일로 저장해 주는 파이썬(Python) 프로그램입니다.

---

## 이 프로그램이 하는 일

1. 한국은행 ECOS API(인터넷 데이터 제공 서비스)에 접속합니다.
2. 내가 지정한 기간 동안의 환율을 가져옵니다.
3. 결과를 `환율_데이터.xlsx` 엑셀 파일로 저장합니다.

수집할 수 있는 통화(화폐):
- 미국 달러 (USD)
- 유럽 유로 (EUR)
- 일본 엔 (JPY)
- 중국 위안 (CNY)
- 영국 파운드 (GBP)

---

## 파일 구성

| 파일 이름 | 설명 |
|---|---|
| `exchange_rate.py` | 환율을 가져와서 저장하는 메인(핵심) 코드 |
| `requirements.txt` | 이 프로그램이 필요로 하는 외부 도구(라이브러리) 목록 |

---

## 시작하기 전에 필요한 것

### 1단계 · 파이썬 설치 확인

터미널(명령 프롬프트)을 열고 아래 명령어를 입력하세요.

```bash
python --version
```

`Python 3.x.x` 처럼 버전이 표시되면 설치된 것입니다.
설치가 안 되어 있다면 https://www.python.org 에서 무료로 설치할 수 있습니다.

---

### 2단계 · 필요한 도구(라이브러리) 설치

터미널에서 이 프로젝트 폴더로 이동한 뒤 아래 명령어를 입력하세요.

```bash
pip install -r requirements.txt
```

> `pip`는 파이썬 도구를 설치해 주는 프로그램입니다.
> `requirements.txt` 안에 적힌 도구들이 자동으로 설치됩니다.

---

### 3단계 · 한국은행 API 키 발급

API 키(프로그램이 데이터를 가져올 때 필요한 비밀번호 같은 것)가 필요합니다.

1. https://ecos.bok.or.kr 에 접속합니다.
2. 회원가입 후 로그인합니다.
3. **"API 서비스" → "API 키 신청"** 메뉴에서 키를 발급받습니다.
4. 발급받은 키를 `exchange_rate.py` 파일 안에 입력합니다.

```python
# exchange_rate.py 파일을 열어서 이 부분을 찾아 수정하세요
API_KEY = "여기에_API_키를_입력하세요"  # ← 발급받은 키로 교체
```

## 사용 방법

### 조회 기간 변경하기 (선택)

`exchange_rate.py` 파일을 열어서 날짜를 원하는 기간으로 바꾸세요.

```python
START_DATE = "20240101"  # 시작일: 2024년 1월 1일
END_DATE   = "20241231"  # 종료일: 2024년 12월 31일
```

날짜 형식은 반드시 `YYYYMMDD` (연도 4자리 + 월 2자리 + 일 2자리)로 입력하세요.

---

### 프로그램 실행하기

터미널에서 아래 명령어를 입력하세요.

```bash
python exchange_rate.py
```

정상적으로 실행되면 아래처럼 출력됩니다.

```
==================================================
  환율 데이터 수집 프로그램 시작!
==================================================
  조회 기간: 20240101 ~ 20241231
  수집 통화: USD, EUR, JPY, CNY, GBP

  📥 미국 달러 (USD) 수집 중...
  ✅ 247개 데이터 수집 완료!
  ...
  💾 엑셀 파일 저장 중... (환율_데이터.xlsx)
  ✅ 저장 완료! '환율_데이터.xlsx' 파일을 확인하세요.
```

완료 후 같은 폴더에 `환율_데이터.xlsx` 파일이 생성됩니다.

---

## 문제가 생겼을 때

| 증상 | 해결 방법 |
|---|---|
| `ModuleNotFoundError` 오류 | `pip install -r requirements.txt` 를 다시 실행하세요 |
| "API 키를 확인해주세요" 메시지 | `API_KEY` 값이 올바른지 확인하세요 |
| 데이터가 0개로 나옴 | 조회 기간에 영업일(평일)이 포함되어 있는지 확인하세요 |
